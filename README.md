<h1>VailDEC3-sources</h1>

<p>このソフトウェアは簡単に使用できることを目標に制作された操作が簡単なファイル暗号化ソフトウェアです。</p>
<p>フリーソフトウェアとして配布され、ソースコード自体も BSD 3-Clause Licenseにてここでオープンソース化しております。

Linuxバージョンは正式版を公開していません</p>
